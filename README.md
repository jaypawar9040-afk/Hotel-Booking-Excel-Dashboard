# Hotel-Booking-Excel-Dashboard
Interactive Excel Dashboard with slicers and charts

## Overview

This project is an interactive Excel dashboard built using a hotel booking dataset. It provides insights into booking trends, cancellations, customer types, and revenue patterns.

## Objectives

* Analyze hotel booking patterns
* Identify cancellation trends
* Understand customer behavior
* Visualize key business metrics

## Dashboard Features

*  Year-wise and month-wise analysis
*  Cancellation rate tracking
*  Customer type segmentation
*  Average Daily Rate (ADR) insights
*  Interactive slicers for filtering

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Graphs
* Slicers

## Dataset

The dataset contains **119,000+ hotel booking records** including:

* Booking status
* Customer details
* Stay duration
* Pricing (ADR)
* Special requests
